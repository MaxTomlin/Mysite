Основные семантические элементы на страницы:

Шапка сайта - header (верхняя часть страницы)

Подвал сайта - footer (нижняя часть страницы)

Блок информации - article (целостный блок информации на странице) Должен содержать заголовки h1-h6

Блок section - объединяет связанные между собой куски информации документа (новости, объединенные по категории) Должен содержать заголовки h1-h6

aside - боковая панель main - основное, не повторяющееся содержимое страницы
